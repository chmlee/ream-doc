Getting Started

The recommended way to edit REAM files it to use REAM Editor, a web-based editor designed to read and write REAM files. It runs entirely in the browser and requires no local installation. A lite version of REAM Editor is embedded in this documentation for readers to test the examples.

(compile output)

You can also edit REAM files with any text editor, and use the standalone compiler to compile the datasets. See reamparser.js and ream-core for more information.
